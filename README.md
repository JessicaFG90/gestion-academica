# 🎓 Gestión Académica — Práctica JavaScript

Práctica de **JavaScript** desarrollada en el módulo de **Lenguaje de Marcas** (1º DAM · Curso 2025/2026).

---

## 📋 Descripción

Aplicación de consola interactiva para gestionar un sistema académico básico. El programa permite registrar estudiantes, añadirles notas y obtener estadísticas de rendimiento, todo desde un menú navegable por teclado.

El proyecto cubrió de forma práctica los conceptos fundamentales de JavaScript: tipos de datos, ámbito de variables, arrays, objetos, funciones, manejo de errores y estructuras de control.

---

## ⚙️ Funcionalidades

- **Registrar estudiantes** con ID, nombre y edad (con validación completa de tipos y duplicados)
- **Añadir notas** a un estudiante existente
- **Calcular la media** de notas de cada estudiante
- **Mostrar información** del estudiante (nombre, edad, número de notas)
- **Listar todos los estudiantes** registrados
- **Menú interactivo** en consola con `readline` de Node.js
